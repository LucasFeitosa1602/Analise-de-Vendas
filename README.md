# Analise de Vendas

# 📊 Análise de Vendas

Projeto de análise de dados focado na exploração e geração de insights a partir de dados de vendas, utilizando Python e bibliotecas de análise de dados.

---

## 🚀 Objetivo

O objetivo deste projeto é analisar dados de vendas e produtos para:

- Entender o desempenho das vendas
- Identificar produtos mais vendidos
- Calcular receita total
- Gerar insights para tomada de decisão

---

## 🛠️ Tecnologias utilizadas

- Python  
- Pandas  
- NumPy  
- Jupyter Notebook  

---

## 📂 Estrutura do projeto

---

## 🔎 Etapas da análise

### 1. Exploração dos dados (EDA)

- Verificação da estrutura dos dados  
- Tipos de colunas  
- Estatísticas descritivas  
- Identificação de valores nulos  

---

### 2. Tratamento de dados

- Limpeza de dados  
- Ajuste de valores ausentes  
- Conversão de tipos  

---

### 3. Integração dos dados

Merge entre:

- vendas.csv  
- produtos.csv  

Permitindo análise mais completa por produto.

---

### 4. Engenharia de features

Criação de métricas como:

Receita = Quantidade * Preco

---

### 5. Análise e Insights

- Produtos mais vendidos  
- Produtos com maior receita  
- Receita total  
- Desempenho de vendas  

---

## 📊 Principais aprendizados

- Manipulação de dados com Pandas  
- Uso de groupby e agregações  
- Merge entre tabelas  
- Criação de métricas de negócio  
- Análise exploratória de dados
